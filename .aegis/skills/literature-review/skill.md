# Skill — Literature Review

## ID

`literature-review`

## Purpose

Compare registered sources, extract themes, identify agreement/disagreement and prepare literature review structures.

## Inputs

- research question
- discipline
- source set
- target structure
- citation style

## Retrieval profile

```text
retrieval_mode: concept_synthesis
requires_evidence_pack: true
source_priority: paper, textbook, lecture_slides, markdown_note
```

## Evidence policy

- Evidence Pack required.
- Agreement and disagreement must be separated.
- Source type and reliability notes must be visible.
- Overclaims must be flagged.

## Output contract

May produce:

- theme map
- source comparison table
- annotated outline
- synthesis paragraphs
- gap analysis
- contradiction report
- reference TODO list
