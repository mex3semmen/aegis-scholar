# AEGIS Schemas

These JSON schemas define the contracts for the research foundation.

Core schemas:

- `source.schema.json`
- `chunk.schema.json`
- `citation-locator.schema.json`
- `evidence-pack.schema.json`
- `retrieval-profile.schema.json`
- `skill.schema.json`
- `audit-event.schema.json`
- `eval-case.schema.json`

Schemas are contracts for implementation. Codex must not invent incompatible fields without updating the schema and related docs.
