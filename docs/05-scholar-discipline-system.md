# 05 — Scholar Discipline System

AEGIS Scholar grows by discipline modules.

## MVP discipline

```text
Psychology + Statistics
```

This matches the first useful academic workflow:

- lecture material
- exercise solving
- APA reports
- method critique
- statistics calculations
- literature summaries

## Discipline profile

A discipline profile defines:

- vocabulary
- concepts
- methods
- output formats
- source quality rules
- common tasks
- retrieval profile defaults
- tool contracts
- evaluation tasks

## Principle

Scientific accuracy comes from:

```text
sources + methods + tools + calculations + evidence packs
```

Not from trusting model memory alone.
